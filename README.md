# Everglow

Free Nitro-inspired Discord themes for BetterDiscord.

> Themes by [EricPanDev / FreeDiscordThemes](https://github.com/EricPanDev/FreeDiscordThemes)

---

## Installation

1. Download the plugin file
2. Move it to:

```txt
BetterDiscord/plugins
```

3. Enable it in:

```txt
Discord Settings → Plugins
```

---

## Usage

- Open plugin settings
- Select a theme
- Use **Refresh** to reload themes
- Use **Unload** to remove the current theme
